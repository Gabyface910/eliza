# eliza
A modern version of the classic '50s chatbot ELIZA.
Eliza is fun to talk to but can be a bit far out and unclear. She is one of the most space-conserving AI models out there, clocking in at a whopping 11 parameters!
The newest version of Eliza is currently `eliza-2-0.000000012b`.
